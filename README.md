# Stringent Constraints on Non-Standard Neutrino Interactions using High-Purity $\nu_{\mu}$ CC  Events in IceCube DeepCore

This repository contains the data files accompanying the paper:

> **"Stringent constraints on non-standard neutrino interactions using high-purity $\nu_{\mu}$ CC events in IceCube DeepCore"**  
> J Krishnamoorthi,  Anil Kumar, Sanjib Kumar Agarwalla
>
We investigate neutral-current non-standard interactions by constraining the parameters $\varepsilon_{\mu\tau}$ and $\varepsilon_{\tau\tau} - \varepsilon_{\mu\mu}$ using publicly available IceCube data, providing most stringent limits on these interaction strengths to date and mapping the allowed regions that remain consistent with observations.


---

## Repository Contents

| File | Description |
|------|--------------|
| **`2D_delta_chi2_map_mutau_magnitude_vs_phase.csv`** | 2D $\Delta \chi^2$ map for the magnitude ($\|\varepsilon_{\mu\tau}\|$) vs phase ($\phi_{\mu\tau}$) components of $\varepsilon_{\mu\tau}$. (Fig. 4 main panel) |
| **`1D_delta_chi2_mutau_magnitude.csv`** | $\Delta \chi^2$ scan of the magnitude ($\|\varepsilon_{\mu\tau}\|$) while profileling over the phase component. (Fig. 4 top panel)|
| **`1D_delta_chi2_mutau_phase.csv`** | $\Delta \chi^2$ scan of the phase ($\phi_{\mu\tau}$) while profileling over the magnitude component. (Fig. 4 side panel) |
| **`1D_delta_chi2_scan_mutau_with_real_value_assumption.csv`** | $\Delta \chi^2$ scan of $\varepsilon_{\mu\tau}$ with a real value assumption. (Fig. 5)|
| **`1D_delta_chi2_scan_tautau_mumu.csv`** | $\Delta \chi^2$ scan of $\varepsilon_{\tau\tau}-\varepsilon_{\mu\mu}$. (Fig. 6)|
| **`global_comparison.md`** | Summary of 90% CL limits on NSI parameters from various neutrino oscillation experiments. (Fig. 1)|
| **`mutau_magnitude_vs_phase_90CL_contour_line.csv`** | Coordinates that outline the 90% confidence contour showing the allowed range of magnitude ($\|\varepsilon_{\mu\tau}\|$) and phase ($\phi_{\mu\tau}$) values. (Fig. 4) |



All the data points are for observed $\Delta \chi^2$ values after profiling over relevant nuisance parameters as described in the paper.
